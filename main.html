Question-1: What are the different ways to select an element in the DOM?

In JavaScript, there are several ways to select elements from the DOM: By id, By class Name, By Tag Name, By Query Selector etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Use innerHTML if you need to insert or retrieve HTML. Use innerText if you only want the visible text. Use textContent for raw text, including hidden elements, and for better performance.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where a parent element listens for events on its child elements, even if those children are added dynamically. Instead of attaching event listeners to multiple child elements, you attach a single event listener to a common ancestor, leveraging event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is a behavior in the DOM where an event triggered on a child element propagates (bubbles) up to its ancestor elements.
